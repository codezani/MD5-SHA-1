# Text Hash Generator

A lightweight, offline Chrome browser extension that instantly converts any text into cryptographic hashes.


## Features

- Real-time hash generation (no need to click any button)
- Supports **MD5**, **SHA-1**, **SHA-256**, and **SHA-512**
- Shows both **lowercase** and **UPPERCASE** versions
- Clean and user-friendly interface
- Works completely offline
- No data is sent to any server (Privacy focused)

## How to Install

1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode**
4. Click **"Load unpacked"** and select the extension folder
5. Done! Click on the extension icon to start using it.

## Usage

1. Click on the extension icon in the toolbar
2. Type or paste any text
3. Hashes are generated automatically in real-time

## Technologies Used

- HTML, CSS, JavaScript
- Web Crypto API
- Pure JavaScript MD5 implementation
